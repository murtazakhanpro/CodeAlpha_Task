# CodeAlpha_Task

A collection of functional, classic Python automation tools and applications developed during the **CodeAlpha Python Programming Internship**. These tasks focus on solidifying fundamental programming mechanics using direct, readable logic.

## 📌 Project Directory & Task Overview

### 🔹 Task 2: Stock Portfolio Tracker (`portfolio_tracker.py`)
A terminal-based financial portfolio calculator. It leverages foundational dictionary operations and input sanitization to manage manual stock positions against a predefined market catalog.
* **Key Concepts:** Dictionaries, looping statements, string alignment formatting, dynamic arithmetic calculation.

### 🔹 Task 3: File Automation Script (`file_automator.py`)
A script designed to streamline file administration. It automatically scans a workspace directory, filters files by file type extensions, and reorganizes them into a dedicated storage layout.
* **Key Concepts:** Core built-in system libraries (`os`, `shutil`), string tracking operations, terminal file IO operations.

### 🔹 Task 4: Basic Chatbot (`basic_chatbot.py`)
An interactive rule-based communication application. It evaluates user message entries against standardized conditions to generate instantaneous contextual replies.
* **Key Concepts:** Sequential execution logic (`if-elif-else`), string transformation processing (`.strip().lower()`), state tracking loops.
